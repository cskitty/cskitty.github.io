# Sorting with C++ struct

C++ struct can be sorted using lamda functions.

{% highlight java linenos %}
struct P {
  int x, y;
};

sort(ants.begin(), ants.end(), [](P a, P b) {return p.x < p.x;});
{% endhighlight %}
